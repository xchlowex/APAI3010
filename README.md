# Academic Project for course APAI3010

## Overview
This repository contains an academic adaptation of the [RealFill](https://github.com/thuanz123/realfill) project.  
The purpose of this work is for **academic work only**.  
We have modified and extended parts of the original code to demonstrate concepts and improvements in image generation.
